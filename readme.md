# Rock Paper Scissors Game

## Description

A Python-based Rock Paper Scissors game where the user competes against the computer using simple game rules.

## Features

* User chooses Rock, Paper, or Scissors
* Computer generates a random choice
* Determines the winner based on game rules
* Displays user and computer choices
* Supports multiple rounds
* Tracks user and computer scores
* Play Again option

## Technologies Used

* Python 3
* Random Module

## How to Run

```bash 
python rock_paper_scissors.py
```

## Game Rules

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock

## Concepts Used

* Variables
* Input/Output
* Conditional Statements
* Loops
* Lists
* Random Module
* Score Tracking
